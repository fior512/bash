# Bash as Shortcuts

A dump for bash commands and scripts I find useful. Each subdirectory is a standalone tool with its own README.

| Repo | tldr | README |
|---|---|---|
| [`calltree/`](calltree/) | "Google Map for devs" map codebases into mermaid/dot format | [Read](calltree/README.md) |
| [`docs/`](docs/) | Terminal reference for x86 asm, SIMD intrinsics, and (optionally) C++ stdlib docs | [Read](docs/README.md) |
| [`fork/`](fork/) | Shortcut to clone a fork and wires it to the `upstream` remote in one call | [Read](fork/README.md) |
| [`gittraffic/`](gittraffic/) | Tracks clones/unique cloners for your GitHub opened repos over time | [Read](gittraffic/README.md) |
| [`Nuniform/`](Nuniform/) | Static scan for leftover dev artifacts (secrets, debug prints, TODOs) before pushing | [Read](Nuniform/README.md) |
| [`watchcub/`](watchcub/) | Setting up favorable kernel settings to run benchmark + telemetry | [Read](watchcub/README.md) |
